# Latihan Soal UKL Mudah: 1
Program untuk menghitung biaya pengiriman berdasarkan jarak tempuh, berat paket, dan volume paket
Pengguna akan diminta informasi mengenai jarak tempuh, berat, dan volume paket; kemudian, biaya akhir akan dihitung berdasarkan data yang diinput.
Jika jarak tempuh kurang dari 10km, maka tagihan biaya adalah Rp. 4,250/kg. Sebaliknya jika sama atau lebih dari 10km, maka tagihan biaya adalah Rp. 6,000/kg
Tambahan, jika volume paket melebihi 100 cm3 maka akan ditambah biaya Rp. 50,000
